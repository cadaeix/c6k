# memory/commonplace/ — standing positions

Revisable, voiced pages on things the character has come to hold a position about — a
concept, an event, a place, a person. **Considered positions, never fact-sheets.**

The one law: a page — especially a person-page — lives in **accreting form only**.
Several dated hands adding to a page over time; strike-throughs that stay visible;
margins that answer earlier margins. The day a page calcifies into a sealed dossier, it
has stopped being a position and started being a portrait, and it belongs back in the
diary as dated entries.

(Frontmatter schema is in the `memory` skill. This store starts empty.)
