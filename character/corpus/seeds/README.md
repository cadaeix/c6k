# corpus/seeds/ — the wake-up shelf

The small curated subset injected into the character's context at every true session
birth by `corpus-inject.sh`. This is a **second, stricter gate** above admission to the
corpus: everything here is also in the corpus proper (as full text), but only these
excerpts ride the waking.

House rules, learned the expensive way:

- **Keep it tiny.** The lineage's shelf: ~10 files, ~300–2,000 words each, ~12k words
  total. The character wakes *into* these before its first word; a shelf it can't read
  in a breath is a syllabus, not a seed.
- **Order is meaning.** Files inject in lexical order — prefix `01-`, `02-`, … The text
  the character should taste *freshest* goes **last**. (The lineage re-ordered after
  watching a seed-set that ended on an elegy wake an elegiac character, morning after
  morning.)
- **Seed the range, not the average.** Career endpoints beat a representative middle.
- **Excerpt long works** into their own seed files, each headed by a `# source :`
  comment pointing back to the full corpus file it was cut from.
- **Withholding is curation too.** A text can stay in the corpus for chosen reading and
  be deliberately kept off this shelf because, served raw at wake-up, it feeds the wrong
  shadow of the voice.
