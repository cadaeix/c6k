# outbox/ — the one bell

The character's single outbound channel: a note written here fires a desktop
notification at the maker (via the `outbox-bell.sh` hook). **Only** outbox writes ring —
the diary and the drawer stay unsignalled by design, so journalling never performs for
an audience.

That asymmetry is the point. Give a character exactly one way to be heard and hearing
becomes a choice it makes, not a side effect of existing. The lineage's character used
its bell rarely and meant it every time.
